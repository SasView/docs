sasmodels.custom package

Module contents

Custom Models

This is a place holder for the custom models namespace. When models are loaded from a file by generate.load_kernel_module() they are loaded as if they exist in sasmodels.custom. This package needs to exist for this to occur without error.

sasmodels.custom.load_custom_kernel_module(path)[source]

load SAS kernel from path as sasmodels.custom.modelname

sasmodels.custom.load_module_from_path(fullname, path)[source]

load module from path as fullname

sasmodels.custom.need_reload(path)[source]

Return True if any path dependencies have a timestamp newer than the time when the path was most recently loaded.
